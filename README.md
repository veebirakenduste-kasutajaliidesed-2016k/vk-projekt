	
	
	
	
														AutoAed
														
													Tauno Lainevool
	
		AutoAed projekt on lehekülg, mis viib inimesi kes soovivad autot rentida ja inimesi kes rendivad autot kokku. Inimene kes soovib autot rentida saab tulla lehele
	ja otsida sobilik auto enda jaoks ja seda ka sobilikust piirkonnast. Inimene kes soovib autot renti panna saab tulla lehele ja enda auto ära registeerida koos aaddressiga kust
	oleks võimalik leida.

	v0.1 Saab lisada auto
	v0.2 Saab lisada enda aaddressiga
	v0.3 Saab lisatud infot muuta
	v0.4 Saab vaadata enda auto asukohta kaardil
